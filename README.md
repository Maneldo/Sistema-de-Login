# Sistema de Login com Flask e SQLite

Sistema web simples de autenticação desenvolvido com Flask, SQLite e bcrypt.

## Funcionalidades

- Cadastro de usuário
- Login com sessão
- Logout
- Página inicial protegida
- Hash de senha com bcrypt
- Validação básica de email
- Banco SQLite criado automaticamente

## Tecnologias

- Python
- Flask
- SQLite
- bcrypt
- HTML
- CSS

## Instalação

Baixe o projeto, extraia o arquivo e entre na pasta:

```bash
cd Sistema-de-Login-Flask-e-Sqlite3
```

Instale as dependências:

```md
pip install -r requirements.txt
```

Execute o projeto:

```bash
python main.py
```

Acesse:
```
http://127.0.0.1:5000
```
